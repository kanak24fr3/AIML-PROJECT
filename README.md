# Student Performance Analysis 📊

## About This Project
This is a beginner-friendly Python project where I analyze student scores using **pandas** and **matplotlib**. The program reads data from a CSV file, calculates total marks and percentages, ranks students, and shows the results with simple charts.

I created this project to practice data handling and visualization, and to understand how to work with real-world data in Python.

---

## What It Does
- Calculates total marks and percentage for each student  
- Ranks students based on their total scores  
- Finds the overall topper of the class  
- Shows a horizontal bar chart highlighting the topper  
- Displays a pie chart of each student’s contribution to the total marks  
- Finds toppers for each subject separately

---

## Dataset
The data is stored in `data.csv` and should look like this:
Name,Math,Physics,CS
Aman,78,82,88
Riya,85,90,92
Raj,60,65,70
Simran,88,85,91
Karan,72,70,68

**Important:** The CSV file must be plain text, not rich text or formatted.
